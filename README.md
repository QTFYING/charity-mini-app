# charity-mini-app

该项目采用uniapp框架开发，功能计划兼容H5和小程序，主要功能面向于慈善会的复购捐。

## 采用技术

* `uni-app` <https://uniapp.dcloud.net.cn/api/>
* `vue3` <https://vuejs.org/>
* `uView v1` <https://v1.uviewui.com/>
